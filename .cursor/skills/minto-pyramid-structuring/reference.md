# Reference: Quality Rubric And Anti-Patterns

## Quality Rubric (0-2 each, total 10)
- **Conclusion clarity**: single, arguable governing thought
- **Logic order discipline**: one order chosen and followed
- **MECE quality**: key points minimally overlapping
- **Evidence mapping**: evidence attached to each point
- **Actionability**: output can drive rewrite or decision

Score interpretation:
- 9-10: executive-ready
- 7-8: usable with minor edits
- <=6: restructure required

## Anti-Patterns To Avoid
- "Conclusion" that is biography or context dump
- Key points that mix causes, outcomes, and actions in one line
- Evidence listed without point ownership
- Repeating the same concept under multiple points

## Fast Repair Tactics
- If conclusion is weak: force template "The core shift is X, therefore Y."
- If points overlap: rename by dimension headers first, then rewrite bullets
- If evidence is messy: tag each evidence line with target point before final output

## Recommended Point Labels
- Cause-Effect mode: Why / Impact / Path
- Structural mode: Market / Capability / Execution
- Time mode: Before / Transition / Next State
