# Reference: STEPPS Scoring Guardrails

## Principle Meanings
- Social Currency: makes sharer look informed/insider
- Triggers: linked to frequent cues or moments
- Emotion: high-arousal feelings that motivate sharing
- Public: observable/socially visible behavior
- Practical Value: concrete utility and use-now value
- Stories: narrative container that carries the idea

## Scoring Guardrails
- Do not treat mere positivity as emotion.
- Do not confuse popularity with practical value.
- Story requires sequence or event framing, not just claims.
- Practical value should include steps/tips/checklist-like utility.

## Confidence Guidance
- High confidence: headline + snippet both show explicit cues
- Medium: cues appear but sparse
- Low: mostly inferred from weak wording

## Rewrite Mapping
- Weak Social Currency -> add insider/first-to-know framing
- Weak Triggers -> bind message to routine moment
- Weak Emotion -> raise arousal (awe/surprise/tension)
- Weak Public -> add social proof/observable behavior
- Weak Practical -> add 3-step utility
- Weak Stories -> add setup -> tension -> resolution
