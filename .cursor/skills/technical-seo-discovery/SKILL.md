# Technical SEO Discovery

## Purpose
Use SEO-oriented discovery rules to improve sourcing quality and reference tags.

## Core Tasks
- Build keyword/topic clusters from current hot topics
- Separate EN and ZH intent spaces
- Prioritize high-intent, actionable terms
- Output frontend-ready hot tags

## Workflow
Task Progress:
- [ ] Collect current hot topics and search demand
- [ ] Cluster into intent groups (problem, solution, comparison, trend)
- [ ] Remove noise and duplicates
- [ ] Publish EN/ZH tag chips to frontend
- [ ] Validate click-through relevance

## Output Requirements
- Show EN/ZH tags separately
- Keep tags concise (2-6 words)
- Include both broad and specific variants
- Keep visual presentation aligned across pages

## Platform Targets
- News: trend + intent clusters
- Podcasts: episode themes + recurring pain points
- Videos: platform-native hot tags + search starters
